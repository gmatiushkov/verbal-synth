# L1 — Структура (оглавление)

> **Уровень 1, наименее детальный.** Только разделы/подразделы. Источник истины по структуре.
> Нижние уровни порождаются из этого: L2 (наполнение в общих чертах) → L3 (углубление) → L4 (текст).
> Меняешь структуру здесь → перегенерируем L2/L3/L4. Методология и статус — в `CONTEXT.md`.
>
> **Решения по структуре (2026-06-16):**
> - синтезатор и интеллектуальный модуль — подразделы единой главы «Разработка программы»;
> - отдельной теоретической главы нет, необходимая теория вплетается по месту;
> - retrieval + модификаторы осей подаются как сразу выбранная архитектура (без хронологии пивотов v2…v5).
